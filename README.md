## Installation

To run, you'll need:
- PHP 8+
- Laragon or XAMPP
- Composer
- VSCode or any other text editor.

To get started:
```sh
cd sixer
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate
php artisan serve
```
